# Pesquisar Paises Ionic

### Consumindo API REST com Ionic 3 e Angular 5 usando HttpClient

#### Para rodar esse exemplo ####

você precisa:
*    nodeJS,
*   Ionic@3 or superior,

### Instale o Ionic



```
npm install -g ionic

```

### Instale Cordova


```
npm install -g cordova

```

### Clone o Projeto
- Clone o repositório

```
$ Run 'npm install'
$ Run 'ionic serve --lab' 

```


### API utilizada  ###

* [restcountries](https://restcountries.eu/rest/v2/)
